# Chat App for Two Users
This project is a real-time chat application designed to facilitate seamless communication between two users. Built with JavaScript and Firebase, it leverages Firebase's real-time database capabilities to ensure instant message delivery and updates, providing a smooth and responsive user experience.


# Key Features:
- Real-Time Messaging: Messages are sent and received in real-time, ensuring that both users can communicate without any delays.
- Firebase Integration: The app is powered by Firebase, utilizing its real-time database to handle data storage and synchronization between users. This ensures that the chat history is always up-to-date, even if one user disconnects and reconnects later.
- User-Friendly Interface: The UI is clean and intuitive, making it easy for users to navigate the app. The design focuses on simplicity and functionality, ensuring that users can focus on their conversation without any distractions.
- Responsive Design: The chat app is fully responsive, meaning it works seamlessly on both desktop and mobile devices. This makes it convenient for users to chat on the go, regardless of the device they are using.
- Secure Communication: Basic security measures are implemented to ensure that the messages between the two users remain private and secure.


  # Technologies Used:
  - JavaScript: The primary programming language used to build the front end and implement the chat functionality.
  - Firebase: Used for real-time database management, ensuring instant data synchronization between users.
  - HTML/CSS: Used to structure and style the application, making it visually appealing and easy to use.
